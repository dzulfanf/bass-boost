# Bass Boost — Monorepo

Monorepo for the Bass Boost product suite.

## Projects

| Folder | Description |
|--------|-------------|
| `bass-boost-extension/` | Chrome extension — bass boost, 10-band EQ, audio enhancer |
| `bass-boost-landing/` | Landing page / marketing site |
| `bass-boost-backend/` | Backend API (planned) |

## General

- All projects use **vanilla JavaScript** — no frameworks, no build tools, no npm dependencies.
- Supported languages: English and Indonesian (i18n).
- Monetization: donation-based Pro tier via Saweria.
- Target platforms: Chrome Web Store (extension), Vercel (landing & backend).
